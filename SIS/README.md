# 📊 Анализатор клиентских отзывов — AI Tool for SME

AI-инструмент для малого бизнеса (ресторан, кафе, сервис), который автоматически анализирует клиентские отзывы и выдаёт структурированный отчёт с рекомендациями.

## Технологии

- **Python** + **Streamlit** — веб-интерфейс
- **Google Gemini 2.0 Flash** — языковая модель для анализа

## Установка и запуск

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Запустите приложение

```bash
streamlit run app.py
```

### 3. Откройте браузер

Приложение откроется по адресу: `http://localhost:8501`

### 4. Использование

1. В боковой панели введите ваш **Google Gemini API Key**
2. Вставьте отзывы клиентов в текстовое поле
3. Нажмите **«Анализировать»**
4. Получите структурированный отчёт

## Как получить API ключ

1. Перейдите на [Google AI Studio](https://aistudio.google.com/)
2. Войдите в аккаунт Google
3. Нажмите **«Get API Key»**
4. Скопируйте ключ и вставьте в приложение
